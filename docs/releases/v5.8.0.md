# Pullora v5.8.0

## Українська

- Додано повну навігацію клавіатурою, помітний фокус і коректне повернення фокуса після закриття діалогів.
- Прогрес, помилки та завершення операцій тепер оголошуються допоміжним технологіям через `aria-live`.
- Інтерфейс підтримує системне налаштування зменшення руху, а контраст усіх тем автоматично перевіряється під час збірки.
- Додано журнал подій із ротацією старих логів і кнопку копіювання діагностики.
- Токени та персональні частини шляхів автоматично приховуються в логах і діагностичному звіті.
- Світлу й темну теми узгоджено, для кожної теми можна вибрати власний фон і налаштувати прозорість та розмиття підкладок.

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

## English

- Added complete keyboard navigation, visible focus, and correct focus restoration after dialogs close.
- Progress, errors, and operation completion are now announced to assistive technologies through `aria-live`.
- The interface respects reduced-motion preferences, and every theme is automatically contrast-checked during builds.
- Added an event log with rotation of old logs and a diagnostics copy action.
- Tokens and personal path segments are automatically redacted from logs and diagnostic reports.
- Light and dark themes are now consistent; each theme supports its own background plus adjustable surface transparency and blur.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
