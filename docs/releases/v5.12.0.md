## Pullora v5.12.0

Реліз завершує безпечну оптимізацію Pullora без зміни прозорої синьої айдентики, геометрії чи поведінки тем.

### Що змінилося

- Прибрано застарілі `.sam-shell`-стилі, непідключені модальні вікна, мертві експорти, події та test-gates.
- Збережено чинні `.cinematic-shell`-компоненти, прозорість, розмиття, синю палітру та однакову сітку світлої й темної тем.
- Settings, About, Install, bulk-дії та великі діалоги завантажуються окремими lazy chunks зі збереженням стану Library і прокручування.
- Український словник залишається стартовим, англійський завантажується окремо; обидва містять однакові 865 ключів.
- GitHub-кеш, черга та обмеження запитів централізовані в Rust; frontend зберігає лише дедуплікацію активних запитів.
- Налаштування вигляду спрощено до реально доступних параметрів щільності, прозорості та розмиття з окремими канонічними палітрами тем.
- Windows release-flow спрощено до підтримуваних EXE, MSI та ZIP; видалено зайві платформні гілки й прямі залежності `tar` та `flate2`.
- Основний JavaScript зменшено до 87,85 КБ gzip, production CSS — до 212,66 КБ / 33,22 КБ gzip.

### Перевірено

- Frontend build, компонентні, i18n і WCAG-перевірки.
- Rust formatting, checks і 23 модульні тести.
- Темна та світла теми, normal/compact, hero/cover/background і клавіатурні сценарії без візуального дрейфу.
- Production Tauri build, release-check та smoke-test portable EXE.

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

---

This release completes Pullora's safe optimization without changing its transparent blue identity, geometry, or theme behavior.

### Changes

- Removed legacy `.sam-shell` styles, disconnected dialogs, dead exports, events, and obsolete test gates.
- Preserved the active `.cinematic-shell` components, transparency, blur, blue palette, and shared light/dark theme grid.
- Settings, About, Install, bulk actions, and large dialogs load as separate lazy chunks while preserving Library state and scroll position.
- Ukrainian remains the startup locale while English loads separately; both dictionaries contain the same 865 keys.
- GitHub caching, request queueing, and rate limiting are centralized in Rust; the frontend only deduplicates active requests.
- Appearance settings now contain only the exposed density, transparency, and blur controls backed by two canonical theme palettes.
- The Windows release flow is limited to supported EXE, MSI, and ZIP formats; unused platform branches and direct `tar`/`flate2` dependencies were removed.
- Main JavaScript is reduced to 87.85 KB gzip and production CSS to 212.66 KB / 33.22 KB gzip.

### Verified

- Frontend build, component, i18n, and WCAG checks.
- Rust formatting, checks, and 23 unit tests.
- Dark/light themes, normal/compact density, hero/cover/background separation, and keyboard scenarios without visual drift.
- Production Tauri build, release checks, and portable EXE smoke test.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
