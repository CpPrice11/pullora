# Pullora v5.7.0

## Українська

- Додано режими бібліотеки «Усі», «Встановлені», «Оновлення» й «Обране», сортування та компактний вигляд.
- Картки показують розмір застосунку на диску й підтримують контекстні дії: запуск, папка, оновлення, версії та видалення.
- Pullora запам’ятовує вибраний застосунок, пошук, фільтри, сортування, щільність і прокручування обох панелей.
- Масове оновлення тепер показує доступне з клавіатури підтвердження зі списком застосунків і версій.
- Великі бібліотеки рендеряться швидше завдяки виміряній оптимізації `content-visibility`.
- Підкладки та користувацькі фони збалансовано так, щоб фон залишався видимим без втрати читабельності.

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

## English

- Added All, Installed, Updates, and Favorites Library modes, sorting, and a compact view.
- Cards show application disk usage and support context actions for launch, folder, update, versions, and uninstall.
- Pullora remembers the selected app, search, filters, sorting, density, and both panel scroll positions.
- Batch updates now use a keyboard-accessible confirmation dialog listing applications and versions.
- Large libraries render faster through measured `content-visibility` optimization.
- Surfaces and custom backgrounds were balanced so artwork stays visible without reducing readability.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
