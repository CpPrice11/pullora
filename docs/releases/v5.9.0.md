# Pullora v5.9.0

## Українська

- Додано множинний вибір застосунків через Ctrl/Shift-клік, Ctrl/Shift+Space, Ctrl+A та Escape.
- Унизу сайдбара з’явилася доступна з клавіатури панель масових дій, яка не перекриває список і підтримує звичайну та компактну щільність.
- Вибрані застосунки можна разом оновити, перемістити до папки, додати до обраного або видалити з нього.
- Масове очищення видаляє тільки неактивні версії, а масове видалення стосується лише локально встановлених застосунків.
- Деструктивні операції показують єдине підтвердження з кількістю застосунків, версій і обсягом даних.
- Після кожної операції показується підсумок успішних, пропущених і невдалих елементів; невдалі залишаються вибраними для повтору.
- У реліз увійшли виправлення читабельності світлої теми та відображення власного фону в бібліотеці, налаштуваннях і розділі «Про застосунок».

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

## English

- Added multi-selection with Ctrl/Shift-click, Ctrl/Shift+Space, Ctrl+A, and Escape.
- Added a keyboard-accessible bulk action panel at the bottom of the sidebar without covering the scrollable list in normal or compact density.
- Selected apps can be updated, moved to a folder, added to favorites, or removed from favorites together.
- Bulk cleanup removes inactive versions only, while bulk uninstall affects locally installed apps only.
- Destructive operations use one confirmation dialog showing app, version, and data-size totals.
- Every operation reports successful, skipped, and failed items; failed items remain selected for retry.
- This release also includes light-theme readability fixes and custom-background visibility across Library, Settings, and About.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
