# Pullora v5.10.1

## Українська

- Повністю відновлено перевірену композицію та оформлення Pullora зі стабільного релізу v5.9.0.
- Повернено попередні розміри, відступи, сітку Library, sidebar, hero card, панелі деталей, налаштування та сторінку «Про застосунок».
- Прибрано новий UI-шар v5.10.0, який спричинив візуальну регресію та небажане перефарбування поверхонь користувацьким акцентом.
- Функціональність масових дій, встановлення, оновлень і локальних даних відповідає стабільній базі v5.9.0.
- Smoke-test portable EXE тепер виконується в ізольованій тимчасовій папці й не залишає тестові файли серед релізних артефактів.

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

## English

- Fully restored Pullora's proven layout and styling from the stable v5.9.0 release.
- Restored the previous sizing, spacing, Library grid, sidebar, hero card, detail panels, Settings, and About page.
- Removed the v5.10.0 UI layer that caused visual regressions and unintended surface recoloring from the custom accent.
- Bulk actions, installation, updates, and local-data behavior match the stable v5.9.0 baseline.
- Portable EXE smoke tests now run in an isolated temporary folder and no longer leave test data among release artifacts.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
