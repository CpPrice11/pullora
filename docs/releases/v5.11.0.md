## Pullora v5.11.0

Реліз завершує уніфікацію основних вікон Pullora без зміни прозорої синьої айдентики та звичного компонування.

### Що змінилося

- Library, Settings, About та Install використовують узгоджену геометрію, поверхні, розмиття й прозорість у світлій і темній темах.
- Загальний фон, hero-фон і обкладинка застосунку залишаються незалежними та не замінюють одне одного.
- Hero card і обкладинка мають однакове положення в обох темах та всіх підтримуваних розмірах.
- Налаштування зберігаються у фоні; локальна кнопка «Скинути» повертає параметри секції «Загальне» до стандартних значень.
- Виправлено перетягування повзунків прозорості й розмиття та їх однаковий вплив на всі робочі області.
- Контекстні, overflow-меню та діалоги використовують спільну клавіатурну навігацію, focus trap, повернення фокуса й обмеження viewport.
- About отримав узгоджені фільтри релізів, статуси та безпечні update/rollback-потоки з обов’язковою перевіркою portable asset і SHA-256.
- Додано відтворюваний parity-gate проти v5.10.1 і розширено release-gate до 19 headless UI-сценаріїв.

### Перевірено

- 48 парних станів Library, Settings, About та Install у темній/світлій темах, українською/англійською, на 1000×700, 1280×720 і 1920×1080.
- Максимальна геометрична різниця відносно v5.10.1 — 0 px; дозволені локальні відмінності задокументовано окремо.
- Frontend build, компонентні, i18n, WCAG і headless UI-перевірки, Rust checks/tests, production Tauri build та smoke-test portable EXE.

Цей реліз не використовує Authenticode Code Signing. Цілісність файлів перевіряється через `SHA256SUMS.txt`.

---

This release completes the unification of Pullora's primary windows without changing its transparent blue identity or familiar layout.

### Changes

- Library, Settings, About, and Install now share consistent geometry, surfaces, blur, and opacity across light and dark themes.
- The global background, application hero background, and cover art remain independent and never replace one another.
- Hero card artwork and cover placement are identical across both themes and all supported viewport sizes.
- Settings save in the background; the local Reset action restores the General section defaults.
- Opacity and blur sliders can be dragged reliably and affect every workspace consistently.
- Context menus, overflow menus, and dialogs share keyboard navigation, focus trapping, focus restoration, and viewport collision handling.
- About includes consistent release filters, statuses, and safe update/rollback flows requiring a portable asset and SHA-256 manifest.
- A reproducible parity gate against v5.10.1 was added, and the release gate now covers 19 headless UI scenarios.

### Verified

- 48 paired Library, Settings, About, and Install states across dark/light themes, Ukrainian/English, and 1000×700, 1280×720, and 1920×1080 viewports.
- Maximum geometry difference from v5.10.1 is 0 px; allowed local differences are documented separately.
- Frontend build, component, i18n, WCAG and headless UI checks, Rust checks/tests, production Tauri build, and portable EXE smoke test.

This release does not use Authenticode Code Signing. File integrity is verified through `SHA256SUMS.txt`.
