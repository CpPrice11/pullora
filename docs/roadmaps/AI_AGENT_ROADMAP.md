# AI Agent Roadmap Pullora

## Статус

Archived / Removed.

AI Workspace більше не є активним напрямом Pullora. Функцію видалено з продукту, навігації, Settings, frontend-сервісів і Tauri backend-команд. Нові роботи в цьому напрямі не плануються в поточному roadmap.

## Рішення

- Основний фокус Pullora повернуто до GitHub Releases launcher і Pullora Store.
- Store, Library, встановлення, оновлення, maintenance і release flow не мають залежати від AI runtime.
- Roadmap не має планувати Codex runtime, workspaces, sessions, approvals або agent extensions як частину Pullora.
- Якщо AI-напрям колись повернеться, його потрібно оформити як новий окремий proposal, а не як продовження цього archived roadmap.

## Що Видалено З Активного Продукту

- Вкладка AI Workspace.
- AI Workspace page і styles.
- Frontend service для AI Workspace/Codex.
- Settings section для AI Workspace.
- Store/Library action `Відкрити в AI Workspace`.
- Tauri commands для AI Workspace.
- Rust Codex runtime adapter.
- AI workspace storage module.
- AI/Codex i18n keys.
